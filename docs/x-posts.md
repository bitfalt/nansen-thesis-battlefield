## Submission-style post (recommended)
Built for the #NansenCLI mini hackathon:

Thesis Battlefield

Paste a thesis or token claim and the system uses the Nansen Query Planner to decide which queries to run under a constrained budget, then returns:
- verdict
- confidence
- strongest evidence
- caveats
- query trace
- JSON output for agents / LLMs

This makes it more than a query wrapper — it’s a reusable intelligence layer on top of raw onchain data.
